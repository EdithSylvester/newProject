CSS Colors
Name Example Description
Color Name Keywords h1 { color: aqua; } Color name keywords can be used to set color property values for elements in CSS
CSS Color Alpha Values .midground { background-color: rgba(0, 255, 0, 0.5);} Alpha values determine the transparency of colors in CSS. Values can be set for both RGB and HSL colors by using rgba() and hsla()
CSS Hexadecimal Colors .red { color: #ff0000;} CSS colors can be represented in hexadecimal (or hex) notation. Hexadecimal digits can represent sixteen different values using 0-9 and a-f.
CSS HSL Colors .light-blue { background-color: hsl(200, 70%, 50%);} CSS colors can be declared with the HSL color system using hsl() syntax. This syntax contains three values: hue (the color value itself), saturation (intensity), and lightness. Hue values range from 0 to 360 while saturation and lightness values are represented as percentages.
